# homebrew-cc-gateway-pro

Homebrew tap for [CC-Gateway-Pro](https://github.com/KeaneFeng/cc-gateway-pro).

## Installation

```bash
brew tap KeaneFeng/cc-gateway-pro
brew install --cask cc-gateway-pro
```

## Update

```bash
brew upgrade --cask cc-gateway-pro
```

## Uninstall

```bash
brew uninstall --cask cc-gateway-pro
brew untap KeaneFeng/cc-gateway-pro
```
